## 解压文件

1. 找到下载好的 ZIP 文件（通常在「下载」文件夹中）
2. **Windows**：右键点击 ZIP 文件 → 选择「全部解压缩...」→ 点击「提取」
3. **macOS**：双击 ZIP 文件，系统会自动解压到同一目录

解压后得到插件文件夹，记住它的位置，下一步需要用到。

## 在 Chrome 中加载扩展

### 第 1 步：打开扩展程序页面

在 Chrome 浏览器地址栏输入以下地址并回车：

```
chrome://extensions/
```

或者通过菜单进入：点击 Chrome 右上角的 **三个竖点 (⋮)** → **扩展程序** → **管理扩展程序**。

### 第 2 步：开启开发者模式

在扩展程序页面的**右上角**，找到「**开发者模式**」开关，将其**打开**（向右拨动）。

开启后页面顶部会出现三个新按钮：「加载已解压的扩展程序」、「打包扩展程序」和「更新」。

### 第 3 步：加载插件

1. 点击左上角的「**加载已解压的扩展程序**」按钮
2. 在弹出的文件选择器中，找到并选中刚才解压得到的插件文件夹
3. 点击「**选择文件夹**」（Windows）或「**打开**」（macOS）

### 第 4 步：确认安装成功

加载完成后，你应该能看到：

- 扩展程序列表中出现「**书签管理器**」
- Chrome 工具栏（地址栏右侧）出现插件图标

> **提示**：如果工具栏没有显示图标，点击工具栏右侧的 **拼图图标 (🧩)** → 找到「书签管理器」→ 点击右侧的 **📌 固定按钮**。

## 配置 Gitee 云同步

安装完成后，建议先配置 Gitee 云同步，这样你的书签可以跨设备同步和备份。

### 第 1 步：注册 Gitee 账号

如果还没有 Gitee 账号，请先前往 [Gitee 官网](https://gitee.com/) 注册一个。

### 第 2 步：创建书签仓库

1. 登录 Gitee 后，点击页面右上角的 **「+」** → **「新建仓库」**
2. 填写仓库信息：
   - **仓库名称**：输入一个名称，例如 `chrome-bookmarks-manage`
   - **是否开源**：选择 **「私有」**（书签属于个人数据）
3. 点击 **「创建」** 完成

> 记住你的 **用户名**（即 Gitee 个人主页地址中 `gitee.com/` 后面的部分）和 **仓库名称**，后面配置时需要用到。

### 第 3 步：生成私人令牌（Token）

1. 打开 [Gitee 私人令牌设置页面](https://gitee.com/profile/personal_access_tokens/new)
2. 填写 **令牌描述**，例如 `书签管理器`
3. 在权限范围中勾选 **`projects`**（项目相关权限）
4. 点击 **「提交」**，输入密码确认
5. 页面会显示生成的 Token，**请立即复制并保存好**

> **重要**：Token 只会显示一次，关闭页面后无法再次查看。如果丢失需要重新生成。

### 第 4 步：在插件中填写配置

1. 点击 Chrome 工具栏中的**书签管理器图标**，打开 Popup 面板
2. 在 Gitee 云同步区域，依次填写：

   | 配置项 | 填写内容 | 示例                        |
   |--------|----------|---------------------------|
   | **Gitee Token** | 第 3 步复制的私人令牌 | `xxxxxxxxxxxxxxx`         |
   | **仓库所有者（Owner）** | 你的 Gitee 用户名 | 如：`zheng_yongtao`         |
   | **仓库名（Repo）** | 第 2 步创建的仓库名称 | 如：`chrome-bookmarks-manage` |
   | **分支（Branch）** | 保持默认即可 | `master`                  |
   | **书签目录** | 保持默认即可 | `bookmarks`               |

3. 填写完 Token、Owner、Repo 后，插件会自动获取仓库的分支列表
4. 配置会自动保存，看到提示「Gitee 配置已保存！」即表示成功

### 第 5 步：首次同步书签

配置完成后，点击 **「覆盖保存」** 按钮，将本地书签上传到 Gitee 仓库。

之后你可以根据需要使用四种同步方式：

| 操作 | 说明 |
|------|------|
| **覆盖保存** | 用本地书签覆盖 Gitee 远程数据 |
| **合并保存** | 将本地与远程书签合并后保存到 Gitee，自动去重 |
| **覆盖获取** | 用 Gitee 远程数据覆盖本地书签（适合在新设备上恢复） |
| **合并获取** | 将远程与本地书签合并后更新本地，自动去重 |

## 安装后的基本使用

### 打开插件面板

点击 Chrome 工具栏中的**书签管理器图标**，即可打开 Popup 面板，包含以下功能入口：

- 打开书签管理器
- Gitee 云同步配置
- 快捷键设置
- 密码保护
- 主题切换

### 全局搜索

在**任意网页**上，连续快速按下 **Cmd** (macOS) 或 **Ctrl** (Windows) 键 **3 次**，即可呼出全局书签搜索框。

- 输入关键词搜索书签标题或网址
- 上下方向键选择结果，回车打开
- 按 **Esc** 关闭搜索框

### 书签管理器

在 Popup 面板中点击「打开书签管理器」，可进入独立的管理页面：

- 左侧为目录树，右侧为书签列表
- 支持搜索、拖拽排序、批量操作
- 提供重复书签检测和失效链接检测

详细的功能说明请参阅 [README 文档](../README.md)。

## 常见问题

### Q：安装时提示「清单文件缺失或不可读取」

请确认你选择的是包含 `manifest.json` 文件的插件文件夹，而不是外层的目录。

### Q：更新插件怎么操作？

1. 从 [插件商城](http://jyeontu.xyz/JYeontuShop/#/plugin/8) 重新下载最新版本并解压
2. 进入 `chrome://extensions/`，找到「书签管理器」，点击 **刷新按钮 (🔄)** 即可

### Q：插件图标在工具栏看不到

点击 Chrome 地址栏右侧的 **拼图图标 (🧩)**，在弹出列表中找到「书签管理器」，点击旁边的 **固定 (📌)** 按钮将其固定到工具栏。

### Q：全局搜索快捷键没有反应

- 确认插件已正常加载（图标显示在工具栏）
- 部分特殊页面（如 `chrome://` 开头的页面、Chrome 网上应用店）不支持内容脚本注入，全局搜索在这些页面上无法使用
- 可以在 Popup 面板的「快捷键设置」中调整触发方式

### Q：支持哪些浏览器？

本插件基于 Chrome Manifest V3 开发，支持以下浏览器：

- **Google Chrome**（推荐，版本 88+）
- **Microsoft Edge**（版本 88+，同样在 `edge://extensions/` 中加载）
- 其他基于 Chromium 的浏览器（如 Brave、Arc 等）

> **注意**：不支持 Firefox 和 Safari。
